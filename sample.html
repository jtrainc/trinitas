About the Club

The Trinitas Code Club is a group of Freshmen students learning to build websites and programs from scratch. Using HTML, CSS, and JavaScript, these students explore how the web works—and how to create their own part of it.

Topics Covered

  • HTML – Structure your content semantically
  • CSS – Design with style and layout precision
  • JavaScript – Make your pages interactive and dynamic
  • Version Control – Learn how to use GitHub like real developers
  • Browser Extensions – Build custom tools for the web

Logic & Programming

Programming is built on the foundation of logic. Just as formal Logic teaches students how to reason with clarity and structure, programming trains the mind to solve problems methodically. Writing code is like creating a set of instructions that must be perfectly logical—computers don’t guess, they do exactly what you tell them.

This class connects classical education with modern skills, allowing students to apply what they know about logical inference, patterns, and structure to build real-world tech.

Student Projects

“Gods of Code” Character Arena

A text-based battle game where each student creates a JavaScript-powered profile inspired by a figure from Greek mythology. The characters face off in a logic-driven tournament.

Responsive Portfolio Pages

Students designed their own web portfolios using Flexbox and Grid, showcasing what they've learned in both structure and design principles.

Browser Extension: Trinitas Tracker

A simple browser extension that gives students reminders, tips, and inspiration as they browse the web.

Why Learn to Code?

Code powers everything from video games and mobile apps to scientific research and financial systems. Learning to code gives students the ability to create, not just consume, the digital world.

Design Matters

CSS isn’t just about color and fonts—it’s about clarity, accessibility, and experience. Students learn how thoughtful design enhances usability and communicates purpose.
